# PySty - Python Styling Library
Enrich your Python Component Library with styles injected via decorators.

#### Easy to use:
{% highlight python %}
def main():
    print("hallo")
{% endhighlight %}

#### Features:
- Easy to integrate
- Supports multiple style themes
- No dependencies
- One decorator and some helper functions
